Introduction


Welcome to the User Guide for the next revolutionary app, THE NFT HUT! This app allows you to manage your NFT collection, analyze it and explore new ones that could be added to your collection. This website allows users to understand how to use the app with the help of the guide and make their experience a satisfying one! The device that we reccomend using with this app is the "Pixel 2 XL"


Login Page


a. Login Event b. Register Event

a. Login Event

The picture shows the display on login to the user. The user is required to input a valid username and password. On successful login, a notification is shown to indicate successful and the user is redirected to the home page. If the details are incorrect, the error is handled and displayed appropriately. To Login, the details are: Username - "steph_curry" and Password - "stephPW"


b. Register Event

When the user clicks the "Register" button, the form for creating a new account becomes visible. The user is required to input First Name, Last Name, Username and Password. On succesful inputs, the user is informed with a notification that the account has been created. Upon the creation of the account, the data is stored in the local sqlite database and Firebase database simultaneously.


Home Page


Upon successful login, the user is redirected to the home page. The home page displays a dashboard containing all the NFTs owned by the user and the total value of the user's collection. The collection of the user is displayed in a gridview which enables the user to scroll through vertically and check their entire collection! Ps: Sometimes not all the assets might show up, to fix this just scroll up and down rapidly and all the assets will appear


a. Home Page b. Clicked Event

b. Clicked Event

Once the user clicks on any of the NFT's, it displays more details about the asset such as it's unique attributes. If the user wants to learn more about the particular asset, they can just click on the "Learn More" button to be redirected to the external link containing in-depth information about the particular asset. If the user wants to share the NFT, they can just click on the share button on the top toolbar to do so.


Explore Page


When the user clicks the respective logo in the bottom navbar, they are redirected to the explore page. This page displays all the latest/in-deman collections for the user to explore. There are 3 rows:

Recently Added: This row shows the user the assets they've recently acquired.

Most Viewed: This row shows the user the Most Viewed assets.

Hot New Items: This row shares with users the most in-demand items that they could be interested in.

a. Explore Page b. Clicked Event

b. Clicked Event

Once the user clicks on any of the NFT's they are interested in, they are shown the asset in detail with its attributes. It is almost identical to the previous clicked event, but here the user can buy these assets as it is not yet added to the user's collection. On click of the "Buy Now" button, the user is redirected to the OpenSea page where the user can buy the particular asset.

Account Page


Once the user clicks the accounts logo, they are redirected to the account page which displays the details of the user. Along with that, the user can logout by clicking the "LOGOUT" button.
